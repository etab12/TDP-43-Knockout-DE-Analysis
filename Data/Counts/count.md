 # Here are the processed data files:

The following files are generated after RNA-seq processing and quantification:

1- gene_counts.tsv

Raw gene-level counts for DEA 

2 sample_info.tsv

Metadata file containing sample IDs and experimental conditions.

3- gene_tpm.tsv

Normalized gene expression values for expression comparison and visualization.

4- tximports.rds

R object containing the gene count matrix.
